# FrutiLogic | Front-End Application

## Summary

FrutiLogic is a specialized inventory management solution designed for fruit store owners and suppliers. Key features include:

- Detailed product care guidelines (storage, handling, and preservation).
- Smart stock tracking for fresh fruits and produce.
- Built-in purchasing system (order directly to suppliers)

## Features

The application includes the following documentation:

- CRUD operations.
- Inventory management.
- Alerts and notifications.
- Catalog and purchase orders.

## Prerequisites

- Node.js (v18 or higher recommended)
- npm or yarn

## Getting Started

1. Clone the repository
2. Install dependencies: `npm install`
3. Configure environment variables (see `.env.example`)
4. Run development server: `npm run dev`
